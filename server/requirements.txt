# QuickPoll

QuickPoll is a full-stack web application that allows users to create, vote, and view polls in real time. Built with a modern stack: **React + Redux Toolkit + TypeScript + Vite** on the frontend, and **FastAPI + MongoDB** on the backend.

## Features

- User registration and authentication (JWT, secure cookies)
- Create, view, and vote on polls
- View latest polls and your own polls
- Responsive, modern UI with Tailwind CSS
- Persistent login with secure session management

## Tech Stack

- **Frontend:** React, Redux Toolkit, TypeScript, Vite, Tailwind CSS
- **Backend:** FastAPI, MongoDB (via Motor), JWT, bcrypt
- **Other:** Axios, Framer Motion, ldrs (loading spinners)

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- Python 3.8+
- MongoDB Atlas account (or local MongoDB instance)
